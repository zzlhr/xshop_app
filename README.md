# xshop_app
![img](https://ws2.sinaimg.cn/large/006tNc79ly1g229v6vzshg30a90kl7wi.gif)
##  功能

- 我的页面
  - 用户信息修改(手机号，用户名只能修改一次，性别，登录密码，头像)
  - 收货地址 (省市区详细地址，收货人，收货人电话，标签)
  - 收藏夹
  - 最近浏览
  - 我的订单
  - 退款
- 注册 (手机号，密码，验证码) 
- 登录(手机号密码，登录手机号验证码登录) 
- 首页
- 购物车
- 通知
- 商品详情
- 商品分类
- 结算
- 版本更新



## 项目结构

- Images 图片文件
- lib dart代码
  - component 公共组件
  - conf 配置文件
  - pages 页面
  - main.dart 启动文件
- ios 自动生成的ios文件
- android 自动生成的Android文件
- test 测试文件夹
